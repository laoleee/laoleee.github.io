梨视频分析报告

发表于 2018-08-15 | 分类于 Project X ， 市场产品分析 | 阅读次数

梨视频是啥

梨视频该产品定位为资讯类短视频，时长多在30秒到3分钟之间，偶有10分钟内的纪录片。

发展历程

  • 2016.07.29 梨视频创始人邱兵从东方早报和澎湃新闻离职，开始短视频项目创业
  • 2016.11.03 梨视频上线，短短几天内就在行业里引起轰动，邱兵的文章刷爆朋友圈，获得近一亿美元的创业资金支持
  • 2017.02.10 梨视频做了第一次升级和转型，将从时政及突发新闻转型为专注年轻人生活、思想、感情等方面，通过讲故事的方式传递中国声音
  • 2017.08底 梨视频宣布与短视频公司Zoomin.TV达成战略合作，联手打造史上最大规模全球拍客联盟
  • 2018.04.16 完成腾讯领投、百度等跟投的6.17亿元人民币A轮融资

一些分析

纵览梨视频的发展历程，短短两年已经发展成为国内一流的短视频平台。一方面感叹其发展之迅速，另一方面也能清晰的感受到这两年互联网内容行业的迅猛发展。关于互联网内容行业，小弟之前写过一篇同属该行业的“一条”的分析文章，各位可以挪步看看。在这还想重申一下我对互联网内容行业为啥能发展得那么快的一点看法：第一是因为消费升级，国民有钱富裕了，开始有余力注重精神方面的体验和享受；第二是因为互联网这个高效的工具对社会各行各业的革命刚好进行到文化领域。

说回梨视频，和一条的创始人一样，梨视频的创始人同样是传统的媒体人出身，同样是转型不转行，其实这种行业内的人跳出来主动高举互联网大旗挑战本行业的事，也是这些年互联网创业的常态，包括我现在的互联网+法律的公司，创始人同样是传统律师出身，成绩也很牛逼，但人家就是敢于挑战自己，挑战自己所处的传统行业。这种做法，在更早一辈眼里看来肯定是接受不了的。可是嘛，总会有一小撮人会提前觉醒，看到未来的趋势，即互联网技术肯定会彻底让整个社会升级换代。

传统媒体行业其实真的没啥出路了，信息传递在今天已经没啥障碍，谁都可以随时随地接受各种新闻资讯，谁都可以做媒体（即自媒体）。至于资讯的内容，普通人并不在乎内容真假，能让人爽就行，传统媒体严肃的老一套在今天并吃不开，一个很简单的例子就是传统新闻资讯的标题要讲究把时间人物地点都说清楚，可是今天的标题主要是要吸引人的目光，让人有点开看下去的欲望。

又扯远了，说回梨视频，只能说梨视频创始人是明智的，能力人脉也是顶尖的，不然谁一开始创业就能搞到一亿美金的资金支持。

梨视频品牌在上线之前也是经过试水市场的，他们在社交平台开设了很多子栏目的账号，例如以人物纪录片为主的“冷面”、关注全球新闻的“时差”、趣味性视频栏目“微辣” “一手”……各种子栏目在app上线前，在微博、微信已经积攒了接近80万的粉丝。完成原始粉丝用户积累后，梨视频上线的时候这些栏目统一换标，全都改为梨视频品牌的下属栏目，华丽转身。对比一条，此处的操作手段是一样的，优质内容积累原始用户，再谋求发展。这也符合互联网企业的发展套路，先抢市场获取用户获取流量再说。

但和一条不同的是，梨视频是铁了心要走内容这条路，那么只能靠持续大量输出高质量的内容才能存活发展。问题是内容从哪里来？这个问题值得好好研究。

我看16年低他们是这样说的：”梨视频的原创分为三类。第一类是纯原创，从拍摄到剪辑等等一系列全部由梨视频完成。第二类叫拍客原创，由我们遍布全球的拍客供稿，梨视频再进行编辑。第三类为编辑原创，题材来自于通讯社和电视台，梨视频对内容进行精简。“

此时梨视频最大的卖点就是其中的新闻拍客爆料专栏。用户可以把自己身边发生的突发性事件用手机拍摄下来，发送到梨视频上，根据播放量平台方提供给拍客一定的奖励。这一机制在运行了两周之后出现了变故。

在2017年2月4日，根据群众举报，北京市网信办、市公安局、市文化市场行政执法总队赴梨视频开展联合执法检查。 经查，由北京微然网络科技有限公司运营的梨视频在未取得互联网新闻信息服务资质、互联网视听节目服务资质情况下，通过开设原创栏目、自行采编视频、收集用户上传内容等方式大量发布所谓“独家”时政类视听新闻信息。依据《互联网新闻信息服务管理规定》、《互联网视听节目服务管理规定》等相关法律法规，网站上述行为属擅自从事互联网新闻信息服务、互联网视听节目服务且情节严重。 （本段来自百度）

这里需要解释一下，国家出于某些方面的考虑，新闻报道这种事是需要领了证才能进行的，没证就干的都是犯法。那么我们平时看的那些搞笑段子，各种身边新鲜事这类内容严格来讲是叫资讯，而不是新闻，不触及法律的前提下谁都可以做。可是新闻和资讯界线确实不那么明显，而梨视频的初始团队大多是传统媒体出身，继续沿用做纸媒的心态和手法来经营着他们的事业倒也能理解，这也是梨视频的视频上很明显的特点，看过都能感受到，和二更、一条的视频，在气质上差别很大。

所以经过此次查办整改后，梨视频CEO邱兵在梨视频官方微信发布《劝君更尽一杯酒，醉了就看梨视频》文章，表示梨视频将从内容上做出较大调整，从时政及突发新闻转型为专注年轻人生活、思想、感情等方面，通过讲故事的方式传递中国声音。 从此开始，平台内容全面升级。从他们的定位也可以看出：资讯类短视频平台。

目前，梨视频的拍客体系，其中包括机构拍客，即自媒体、媒体及入驻机构（人民日报、办公室小野等）；核心拍客群，指普通拍客（用以保证梨视频素材的宽度）；拍客plus，指国家地理杂志等业内非常专业的媒体人（拍摄完的视频，不能修改，用以保证梨视频内容的深度）；商业拍客，指接受过专业培训的拍客或者兼职媒体人。上述拍客网络体系是梨视频视频内容的重要来源。

至此，可以解决这个问题了，内容哪里来？主要是拍客来提供。

那么如何保证内容的质量呢？

机构拍客、拍客plus、商业拍客这些拍客都是专业生产视频内容的，内容质量问题无需太过担心。重点是如何把控普通拍客的内容质量？梨视频是这样干的，一条视频想要播出来，需要经过这些步骤：第一普通用户将视频拍摄后上传视频内容到梨视频的后台；第二是由本地主管进行审核；第三则由统筹部门来审核视频内容，通过后则发给剪辑进行审核；第四剪辑完后发给领导，领导审核后才能发布。 所以，普通拍客只是提供素材的。

梨视频为了内容质量采用了最复杂成本最高的方式，即人工层层审核，所以当你看到了梨视频某条看似平常、只有几十秒钟的视频内容，其实它已经经过各种人的手，经受了反复打磨。由于新闻和资讯界线不明显的问题，梨视频由为了做资讯，免不了有严格审核机制，同时为了拿到独家内容，自己不能支撑过于庞大的采编团队，只能通过招募拍客的形式进行发展，从而生产内容，但是又需要对内容质量负责，所以采用自己编辑视频，自己层层审核的方式。

当视频数量开始增大后，审核和剪辑会消耗巨大的成本，所以如何降低这里的成本又成了一个问题。梨视频打算利用技术来解决这个问题（用户个性化推荐、与美国一家智能剪辑公司合作等事件），但是目前算法并不知道如何找出符合社会主义核心价值观的视频，算法也不知道如何剪辑出矛盾突出，直击本质的视频。所以梨视频利用技术来降低审核剪辑成本的想法只能慢慢摸索，暂时无解。

梨视频计划在2018年将核心拍客的队伍扩充到5万人，此数据在17年8月份是1.8万。所以在今年4月份的融资大头肯定是放在拍客招募管理上。

随着拍客人数的增加，如何管理拍客与视频内容又成为一个问题？

2017年11月16日，邱兵在一场演讲中介绍了他正在推动的用于管理拍客的Spider系统。 该系统具备从素材采集到支付以及最后编辑上线的全流程响应机制。在全球范围内，Spider系统能做到30分钟响应；在国内，Spider系统能做到中国2000个县市同步响应。

对于大多数公司，做一个自己的内部系统是挺好的，就是当下成本有点高，属于功在当代，利在千秋的事，主要是能不能活到千秋，成本考虑值不值得，这个就要公司自己考量了。

最后要谈的是商业化问题，即怎么盈利，怎么发展壮大的问题。

梨视频的目标是，做一个资讯短视频平台。手段是，靠给普罗大众发补贴的形式发动他们走向街头，发掘身边值得报道传播的人物、故事。但梨视频为了保证只做资讯、保证内容质量，养了一个巨大的审核编辑团队。

假设已经有了拍客拍下的素材了，靠谱内容公司也能实现低成本量产，产出来的内容也有人看了。那么公司如何赚钱？我们知道，互联网平台一般靠广告挣钱。可是梨视频很难成为一个比较大的平台，因为它只生产内容，它并不是渠道分发商，没有自带流量那种属性。它的视频大多数在微博、微信这些地方进行分发，用户只是把它当作一个自媒体看待，并不会主动下载它的App，更不要说作为使用App的活跃用户了。所以梨视频想做平台，但是怎么样让用户进入自己的生态体系还需考虑。无论是发广告还是让用户付费，有了用户才可以收割用户，这是互联网行业的基本玩法。

刚刚说的是收入问题，现在来谈支出问题。梨视频自己不养采编团队，可是却养了几万的普通拍客，养了一支庞大的审核编辑团队，这对企业来说是一笔不小的支出。

这么一说，梨视频靠目前的模式赚大钱是比较难的。但是只要能将用户从其他渠道引流到自己的生态体系里，那这事就能玩。

最后

祝梨视频能走出一条自己的路。
