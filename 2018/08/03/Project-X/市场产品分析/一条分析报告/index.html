一条分析报告

发表于 2018-08-03 | 分类于 Project X ， 市场产品分析 | 阅读次数

企业简介

一条是一家主打生活短视频的互联网新媒体，创办人为外滩画报前总编徐沪生。一条创办于2014年9月8日，首先在微信开办公众号（订阅号），以每天一条的节奏，发布原创短视频。 杂志化的视频，是一条明显区隔于其他电视节目与视频节目的特点。其镜头缓慢，趋于静态，强调布景与摆设。选题多采用中产阶级的人物为主角，表现其特定的某一类生活方式。定位高端，不做幽默的段子，也不做接地气的UGC。一条分为多个栏目，接近杂志的设定。每个栏目或为周播，或为双周播。虽然一条整体节奏为日播，但从栏目结构来看依然是周播节目。在移动互联网视频里面，一条专注于生活类视频。包括：生活（美食、酒店、汽车等）、潮流（女性时尚、男性时尚、美容等）、文艺（建筑、摄影、艺术等）。

（注：以上介绍来自百度百科）

一点个人分析

互联网内容行业的发展

纵览一条的发展历程，用顺风顺水、势如破竹来形容也不为过。15天粉丝破100万这种速度确实有点可怕，查看报道资料，这个战绩一条也是花了些广告费才得来的，但是如果内容不好，花再多的钱也买不来用户，买得来也留不住。所以透过一条，我们可以清晰的感知到互联网内容行业的蓬勃发展。

再究其原因，互联网内容行业的发展是其实是国民消费升级的一种表现，即人们开始注重生活品质，开始注重精神生活的享受。

其实互联网这个物种从九十年代传入中国后就从各个方面开始影响人们的生活，从最早的Web1.0，以资讯门户网站为主，人们开始认识互联网，学着使用互联网，从互联网上了解资讯信息，第一次实实在在感受到了互联网的便利。此时网上的内容，多是商业机构来主导产生，国外的雅虎，国内的新浪、搜狐等都是这个时期的代表。

Web2.0时期，人们已经认识并开始接受互联网这个物种，甚至已经有点离不开它所带来的便利，此时的网民开始在网上聊天交友，发表各种看法。各种聊天室、论坛、博客是这时的主流，此时网上的内容，多是以用户产生为主，互联网的进化开始以普罗大众的推动为主导。如果说Web1.0是让人们认识互联网，那么Web2.0就是人们开始和互联网交流。

随着iPhone的发布，移动互联网大潮席卷而来，这十年来，网速越来越快，数据流量费越来越便宜，这使得人们能更加方便能够接触互联网，这也使得一个个行业被互联网的高效便捷革了命，电商、在线教育、外卖、出行等等，例子太多太多，此时互联网上的内容差不多和现实生活同步了。人们一半活在线上，一般活在线下。

而未来，“互联网+”依然高举大旗，誓死要革传统行业的命，人们的生活将更加彻底融入互联网；另一方面，“物联网”蓬勃发展，人们已经不满足与人与机器交流了，需要机器与机器也进行交流。

但是，也正是这样飞速发展的社会背后，精神生活是有所缺失的。社会人心越浮躁，人们越渴望精神力量的回归。其实你应该也发现了，超过140字的文字读不下去了，音乐满脑子都是抖音神曲，网友见面都不知道说啥，周末的活动范围仅仅卧室那几平米，等等。

有时候我们都会问自己，社会越来越便捷的背后到底给我们带来了什么。如果此时我们在网上看到，在这个浮躁的气氛里面仍然有人在坚守一些东西，那我们一定是有所触动的。今天所谓好的内容，和古人推敲诗句干的是同样的事，大家都希望在时代留下些不一样的东西，古人吟诗作对，今人在网上写图文，拍视频。

只是互联网所掀起的社会变革刚好进行到艺术创作领域罢了，你看吧，基本的衣食住行、社交、娱乐都被互联网满足大众之后，那下一步必然是更高层次的精神享受领域。好的文章，好的视频都是一种享受，能发人深思，给人触动。

这么一分析，再看互联网内容行业的发展，其实也就自然而然了。

一条的发展分析

一个行业发展起来，那必然会出现一批靠着优质内容产品存留下来的企业，一条正是其中之一。一条瞄准的是中产阶级，这个阶级不算缺钱，懂生活享受，一条生产的优质视频内容能深受这个群体的喜爱，这是很自然的事。

一条的模式是这样，自己生产优质内容，靠各大内容分发渠道进行传播，包括微信、优酷、腾讯视频、微博等，从各大渠道网罗一大批优质的用户后，那之后的事就稍微容易了，靠拍优质广告是一种活法，靠用户对视频的内容的兴趣，做电商生意也是一种活法，江湖人称内容电商。一条目前是两条路都在走，而且走得都异常平稳。

看了下一条生活馆卖的东西，不算便宜但也不算贵，标准的小资生活所需内容。一条生活馆已经是一条最主要的利润来源，按照创始人的想法，视频广告主要任务是养活公司，一条生活馆是盈利来源，按照他这个规划和去年12月份数据，那它现在活得那是相当滋润的。

电商这条路持续么？看一条的数据可以发现，其实靠视频网罗的用户其实已经到了瓶颈期，市场并不会无限制的大下去。虽然靠这部分用户已经能让公司上下活得很滋润，但它的电商之路也就到此为止了。

一条需要更广的获客渠道。从年初的融资开始，一条就在建立自己的线下生活馆，计划为城市的中产阶级们提供更好更有温度的线下服务。其实这也是互联网手机公司的老套路，线上你能影响的用户有个总量，过了这个量再努力也没用了，需要转入线下渠道，小米就是个典型。但是线下门店可是重资产投入，怎么控制成本就看一条的能力了。

总结

一条作为互联网内容行业的杰出代表，本文从社会发展和商业模式上对其进行了分析讨论。最后我认为互联网内容行业其实还有很多潜力可以挖掘，一条的内容电商之路仅仅为其中一种可能。毕竟社会、大众对优秀内容是需求旺盛的，只要这个需求在，这个行业就永远有机会。
